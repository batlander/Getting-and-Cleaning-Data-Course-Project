# Code Book

Resume of the `tidy.txt` data.

## Variables

* Subject - ID of the person tested
* Activity - Type of activity tested

## Type of Activity

* WALKING (number 1)
* WALKING_UPSTAIRS (number 2)
* WALKING_DOWNSTAIRS (number 3)
* SITTING (number 4)
* STANDING (number 5)
* LAYING (number 6)
